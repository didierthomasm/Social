## 📥 Installation & Setup

### 1. Clone the Repository

To clone the repository, use the following command:

    git clone https://github.com/didierthomasm/Social.git

### 2. Navigate to the Project Directory

    cd Social

### 3. Install Dependencies

    npm install

### 4. Start the Server

    npm start
